## Goal
Add SVG snapshot tests using `pytest-textual-snapshot` for visual regression testing. These tests are **Windows-only** — skipped on other platforms to avoid baseline drift.

## WHERE — Files

### New files
- `tests/icoder/test_snapshots.py`

## HOW — Integration Points

- `pytest-textual-snapshot` provides the `snap_compare` fixture
- `snap_compare` takes an `App` instance, runs it headlessly, captures SVG
- First run generates baseline SVGs in `snapshot_tests_output/`
- Subsequent runs compare against baselines
- `--snapshot-update` flag regenerates baselines
- Tests use `FakeLLMService` to produce deterministic output for stable snapshots

### Snapshot test pattern

```python
async def test_initial_empty_state(snap_compare):
    app = create_test_app()  # ICoderApp with FakeLLM
    assert snap_compare(app)
```

For tests requiring interaction before snapshot:

```python
async def test_after_help_command(snap_compare):
    app = create_test_app()
    async with app.run_test() as pilot:
        await pilot.type("/help")
        await pilot.press("enter")
        await pilot.pause()
    assert snap_compare(app)
```

## DATA — Return Values

- `snap_compare()` returns `bool` — True if SVG matches baseline
- SVG files stored in `tests/icoder/snapshot_tests_output/`

## Tests — `tests/icoder/test_snapshots.py`

```python
import sys
import pytest
from mcp_coder.icoder.core.app_core import AppCore
from mcp_coder.icoder.core.event_log import EventLog
from mcp_coder.icoder.services.llm_service import FakeLLMService
from mcp_coder.icoder.ui.app import ICoderApp

pytestmark = pytest.mark.skipif(
    sys.platform != "win32",
    reason="Snapshot tests are Windows-only to avoid baseline drift"
)


def _create_app(tmp_path) -> ICoderApp:
    """Create ICoderApp with FakeLLM for deterministic snapshots."""
    fake_llm = FakeLLMService()
    event_log = EventLog(logs_dir=tmp_path)
    app_core = AppCore(llm_service=fake_llm, event_log=event_log)
    return ICoderApp(app_core)


# Snapshot: initial empty state
async def test_snapshot_initial_state(snap_compare, tmp_path):
    app = _create_app(tmp_path)
    assert snap_compare(app)


# Snapshot: after /help command
async def test_snapshot_after_help(snap_compare, tmp_path):
    app = _create_app(tmp_path)
    async with app.run_test() as pilot:
        await pilot.type("/help")
        await pilot.press("enter")
        await pilot.pause()
    assert snap_compare(app)


# Snapshot: after a sample conversation (user input + fake LLM response)
async def test_snapshot_after_conversation(snap_compare, tmp_path):
    app = _create_app(tmp_path)
    async with app.run_test() as pilot:
        await pilot.type("What is Python?")
        await pilot.press("enter")
        await pilot.pause(delay=0.5)
    assert snap_compare(app)
```
