feat(langchain): extract token usage from text and stream paths (#819)

Add _usage.py submodule with _extract_usage() and _sum_usage() helpers that
map LangChain's usage_metadata (with nested input_token_details) onto the
provider-agnostic UsageInfo TypedDict.

Wire _extract_usage() into:
- _ask_text(): populates raw_response["usage"] from chat_model.invoke() result
- _ask_text_stream(): tracks any chunk with non-empty usage_metadata (last-wins)
  and emits the extracted usage in the final "done" event

The helpers live in a dedicated _usage.py submodule (rather than __init__.py)
to avoid circular-import risk with agent.py, which will consume _sum_usage in
the following step.
