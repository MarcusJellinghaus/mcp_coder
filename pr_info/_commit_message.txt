feat(langchain): sum token usage across agent steps (#819)

Accumulate usage_metadata from every AIMessage in run_agent() stats and
every on_chat_model_end event in run_agent_stream() done event, using
_extract_usage and _sum_usage helpers from the _usage submodule.

- run_agent(): sums usage across all AIMessages in output and exposes
  it in stats["usage"].
- run_agent_stream(): listens for on_chat_model_end events carrying
  the full AIMessage with usage_metadata, sums across events, and
  emits the accumulated total in the final "done" event.
