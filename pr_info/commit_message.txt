Delete testdata/ folder (#820)

The testdata/ folder was used as visual acceptance tests during the
redesign of tool-output rendering. With the rendering expectations now
encoded in unit tests (test_stream_renderer.py) and the new formatter
pipeline in place, the JSON fixtures, rendered outputs, and one-off
development scripts (render_testdata.py, render_nice.py) are no longer
needed.

Removed:
- Rendering scripts (2 files)
- Design document (1 file)
- Non-tool test data (4 JSON files)
- Tool JSON inputs (14 files)
- Rendered outputs (42 files: 14 x 3 variants each)

No production code imports from testdata/; all tests still pass.
