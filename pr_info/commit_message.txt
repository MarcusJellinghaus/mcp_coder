feat(langchain): run_agent_stream accepts pre-built tools (#741)

Add optional `tools` parameter to `run_agent_stream()`. When pre-built
LangChain tools are provided (e.g. from MCPManager), the function skips
MultiServerMCPClient creation and uses the provided tools directly.
When `tools=None` (default), existing behavior is unchanged.

Add tests verifying both paths:
- Pre-built tools bypass MCP client instantiation
- Default path still creates MCP client as before
