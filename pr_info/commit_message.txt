feat(icoder): pass project_dir to RealLLMService for prompt injection

Add project_dir parameter to RealLLMService and pass it through to
prompt_llm_stream() in the stream() method. iCoder always injects
system prompts by passing project_dir when creating RealLLMService
in execute_icoder().

Changes:
- RealLLMService.__init__ accepts optional project_dir parameter
- RealLLMService.stream() forwards project_dir to prompt_llm_stream()
- execute_icoder() passes project_dir=str(project_dir) to RealLLMService
- Added tests verifying project_dir pass-through and None default
