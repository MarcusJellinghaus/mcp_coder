feat(langchain): thread tools param through _ask_agent_stream and ask_langchain_stream (#741)

Add optional `tools: list[Any] | None = None` parameter to both
`_ask_agent_stream()` and `ask_langchain_stream()` in the langchain
provider __init__.py. Each function passes it through to the next layer
(`run_agent_stream`), enabling pre-built MCP tools from MCPManager to
bypass MultiServerMCPClient creation.

- `_ask_agent_stream()` gains `tools` param, forwards to `run_agent_stream()`
- `ask_langchain_stream()` gains `tools` param, forwards to `_ask_agent_stream()`
- Text mode path unaffected (tools param ignored when no mcp_config)
- Tests: verify tools forwarded to run_agent_stream, tools=None by default,
  text mode works with tools param present
