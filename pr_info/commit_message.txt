Add checkout-issue-branch subcommand to gh-tool

Implement `mcp-coder gh-tool checkout-issue-branch <number>` which
fetches issue data, finds or creates a linked branch via GitHub API,
and checks it out locally. Exit codes follow the existing pattern:
0 (success), 1 (branch not found/created), 2 (error).

- Register checkout-issue-branch subparser in parsers.py
- Wire dispatch and import in main.py
- Add unit tests (exit codes, behavior) and CLI integration tests
