feat(icoder): add autocomplete state module

Add pure autocomplete state module with no Textual dependency:

- AutocompleteState frozen dataclass capturing visible, matches, and
  highlighted_index
- compute_next_state() pure function that derives state from input text
  and command registry
- Dropdown visible whenever input starts with '/', even with no matches
- highlighted_index is 0 when matches exist, -1 when empty

Add 7 unit tests covering visibility, filtering, highlighting, and
frozen immutability invariants.
