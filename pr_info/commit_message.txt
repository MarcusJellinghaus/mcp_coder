icoder: langchain text stream inactivity timeout

Add inter-chunk inactivity timeout to _ask_text_stream() so that a
stalled LangChain model connection raises TimeoutError instead of
hanging forever. The timeout check runs before processing each chunk,
with last_activity initialized before the loop to avoid false timeouts
on slow initial connections.

Also add a `except TimeoutError: raise` clause before the general
exception handler to prevent TimeoutError (a subclass of OSError)
from being misclassified as a connection error.
