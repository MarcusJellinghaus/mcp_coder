feat(icoder): add /info slash command with runtime diagnostics (#741)

Add `/info` command that displays:
- mcp-coder version, Python version/path
- Environment paths (tool env, project env, project dir)
- MCP server status (langchain and Claude Code)
- Environment variables with secret redaction

New files:
- src/mcp_coder/icoder/core/commands/info.py
- tests/icoder/test_info_command.py

The `_redact_env_vars` helper redacts values for keys containing
token, key, secret, password, or credential (case-insensitive).
