feat(tui): add VS Code gpuAcceleration detection check

Add _check_vscode_gpu_acceleration() to TuiChecker that detects when
VS Code terminal has gpuAcceleration set to 'off', which breaks TUI
mouse/rendering.

Gates:
- Windows only (sys.platform == "win32")
- Skips if SSH_CONNECTION is set (remote session)
- Only runs when TERM_PROGRAM == "vscode"

Uses regex search on %APPDATA%\Code\User\settings.json to detect the
setting without JSON/JSONC parsing. When detected, appends a prompt
that gives the user Instructions or Abort options (both abort).

Wired into run_all_checks() between codepage fix and warning checks.
Includes 9 tests covering all gates, detection, and prompt flows.
