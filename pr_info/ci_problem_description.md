The CI pipeline fails with three distinct issues, all stemming from the formatter consolidation refactoring (issue #737) that moved formatting from `mcp_coder.formatters` to `mcp_tools_py`.

The primary unit test failure is `TestRunImplementWorkflow.test_run_implement_workflow_success` in `tests/workflows/implement/test_core.py:1159`. The test mocks the high-level workflow functions (`check_git_clean`, `check_main_branch`, `check_prerequisites`, `prepare_task_tracker`, `process_task_with_retry`) but does NOT mock the final mypy check and formatting steps that run after task completion (lines 660-696 in `src/mcp_coder/workflows/implement/core.py`). When `process_task_with_retry` returns `(True, "completed")` the workflow proceeds to the "final mypy check" block, which calls `check_and_fix_mypy()` with the fake path `/test/project`. This hits the real `run_mypy_check` which raises `FileNotFoundError: Project directory not found: /test/project`. Subsequently, `run_formatters()` fails with `No module named 'mcp_tools_py.formatter.runner'` because the import path changed or the installed `mcp_tools_py` package version in CI does not include the formatter module. The workflow returns 1 instead of the expected 0. The fix requires adding `@patch` decorators for `check_and_fix_mypy` and `run_formatters` to the test so these post-completion steps don't execute against a non-existent directory.

The second failure is a test collection error on `tests/test_mcp_tools_py_integration.py`, which uses the `@pytest.mark.formatter_integration` marker on line 14. Step 2 of the implementation plan removed the `formatter_integration` marker definition from `pyproject.toml`, but this test file still references it. Since the marker is no longer registered, pytest raises `'formatter_integration' not found in markers configuration option` during collection. The fix is to either restore the marker in `pyproject.toml`, replace it with an appropriate registered marker in the test file, or remove the marker entirely if the test class should use a different categorization.
